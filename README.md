# StoryForge
Here's an open-source Launcher+Mod Loader just for MCSM! Made by (@B0Zin0)

#What it can do.
Fully Supports mods with ant telltalle files such as dialoge changes.
Fully Supports auto plugin mods into your main game.
Fully Supports the MCSM Season 1, and 2. 
Fully Supports Audio and Video (Mp3,Mp4)

#Sadly...
Modding MCSM can take up many years due to it not really being "modable" *Telltale fault*
However it is easy to change character skins and plenty more features i'd suggest to search it up however, I used TellTale explorer!
